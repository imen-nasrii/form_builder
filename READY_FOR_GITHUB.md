## 🔧 Local Setup for Contributors

### React Architecture
```bash
git clone https://github.com/yourusername/formbuilder-pro.git
cd formbuilder-pro
npm install
cp .env.example .env
# Configure DATABASE_URL and API keys
npm run dev
```

### .NET Architecture
```bash
git clone https://github.com/yourusername/formbuilder-pro.git
cd formbuilder-pro
dotnet restore
cp .env.example .env
# Configure connection strings
dotnet watch run
```
