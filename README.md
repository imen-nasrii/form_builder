## 🎯 Project Capabilities

### Dual Architecture Support
**Choose your preferred technology stack:**

#### Option A: Modern Web Stack (React/Express.js)
```bash
git clone https://github.com/imen-nasrii/formbuilder-pro.git
cd formbuilder-pro
npm install
npm run dev  # Starts on http://localhost:5000
```

#### Option B: Enterprise Stack (.NET Blazor)
```bash
git clone https://github.com/imen-nasrii/formbuilder-pro.git
cd formbuilder-pro
dotnet restore
dotnet watch run  # Starts on https://localhost:7000
```
